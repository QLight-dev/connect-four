# Connect Four
A CLI-based Connect Four game written in pure Go.
It has a clean, beautiful board complete with a working input and win detection system.

![Connect Four Demo](assets/demo.gif)

## Installation
### Using ```go``` (Recommended)
> [!NOTE]
> requires ```go 1.25.7+```, if you have a lower version, you should one of the other methods below
```zsh
go install github.com/QLight-dev/connect-four
connect-four
```
### Using ```git```
```zsh
git clone https://github.com/QLight-dev/connect-four.git
cd connect-four
./connect-four
```
### Using ZIP
1. Navigate to [https://github.com/QLight-dev/connect-four](https://github.com/QLight-dev/connect-four).
2. Click on "Download ZIP".
3. Open your downloads page in your browser, there will be your zip.
4. Extract the ZIP.
5. Open the folder and double click on the ```connect-four``` binary.
