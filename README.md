# connect-four
connect-four is a CLI Connect Four app is entirely written in Golang. It has visually-pleasing asthetics, a complete input detection system, all in your terminal
## Installation
### Using Go
```zsh
go install github.com/QLight-dev/connect-four
connect-four
```
### Using git
```zsh
git clone https://github.com/QLight-dev/connect-four.git
cd connect-four
./connect-four
```
